# SW2.5-Dice-Buff-Helper
ソードワールド2.5TRPGに対応しています。 威力表によるダメージ算出、命中力、回避力のそれぞれについて、バフの管理やダイスコマンドとしての出力が可能です。<br>

#セーブ・ロードについて
----
　ダメージ、命中、回避タブについて、個別にセーブ・ロードできます。<br>
　異なる種類のセーブデータはロードできません。<br>
<br>
　データ管理タブから、すべてのタブをまとめてセーブ・ロードできます。<br>
　個別ロードは全体セーブデータをロードできません。逆もしかりです。<br>
<br>
　ファイルからセーブ・ロードでは、セーブデータのファイル出力、およびファイル読み込みができます。<br>
　テキストからセーブ・ロードでは、テキストフィールドにセーブデータの出力や、テキストフィールドに入力したセーブデータを読み込むことができます。<br>
　データテキストコピーから、テキストフィールドのセーブデータをコピーできます。<br>
 <br>
※ファイルからのセーブ・ロードは、一部ブラウザ(safari、firefoxなど)では対応していません。テキストからのセーブ・ロードをご利用ください。<br>
<br>

#入力欄について
----
　武器名には、武器名を入力します。威力値・クリティカル値・固定値・命中力・回避力の入力欄には、バフを含んでいない場合の値を入力します。<br>
<br>
・敵防護点あり:チェックを入れると敵防護点の欄が出現し、敵防護点を含めた出力ができます。<br>
・半減:ダメージを半減します。(攻撃魔法の抵抗成功時など)<br>
・クリティカル無し:クリティカル値が13となり、クリティカルが発生しなくなります。<br>

#バフ枠・ダイス種類について
----
　各タブの+と-で、バフ枠を増減できます。<br>
　効果名にバフの名前を、効果に数値の変動量、ダイス効果(ダメージのみ)に効果の種類を入力します。チェックボックスで、バフの有効化、無効化を切り替えられます。<br>
<br>
・固定値:数値の固定値を増減します。(マッスルベアー、全力攻撃など)<br>
・C値増減:クリティカル値が増減します。(かいくぐり、クリティカルキャストなど)<br>
・ダイス目増減(継続):威力値の出目が増減します(必殺攻撃など)<br>
・ダイス目増減(初回):威力値が増減します。クリティカル後に適用されません。(クリティカル・レイなど)<br>
・ダイス目固定:片方のダイス目を固定します(薬師道具セットなど)<br>
                            <br>
　上矢印・下矢印のボタンでバフ欄を入れ替えられます<br>

#表示/コピーについて
----
　各タブの表示コピーで、ダイスコマンドを表示、およびコマンドをクリップボードにコピーします。そのままダイスコマンドとして貼り付けて使用可能です。<br>
<br>
・個別出力:固定値・C値・命中力・回避力を、合算して出力する(例:2d6+6)か、個別に出力する(例:2d6+5+1+1-1)か選択できます<br>
