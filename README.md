# SW2.5-Dice-Buff-Helper
ソードワールド2.5TRPGに対応しています。 威力表によるダメージ算出、命中力、回避力のそれぞれについて、バフの管理やダイスコマンドとしての出力が可能です。

#セーブ・ロードについて
----
ダメージ、命中、回避タブについて、個別にセーブ・ロードできます。
異なる種類のセーブデータはロードできません。

データ管理タブから、すべてのタブをまとめてセーブ・ロードできます。
個別ロードは全体セーブデータをロードできません。逆もしかりです。
----
#入力欄について
----
武器名には、武器名を入力します。威力値・クリティカル値・固定値・命中力・回避力の入力欄には、バフを含んでいない場合の値を入力します。敵防護点あり:チェックを入れると敵防護点の欄が出現し、敵防護点を含めた出力ができます。                           

#バフ枠・ダイス種類について
----
各タブの+と-で、バフ枠を増減できます。
効果名にバフの名前を、効果に数値の変動量、ダイス効果(ダメージのみ)に効果の種類を入力します。チェックボックスで、バフの有効化、無効化を切り替えられます。

固定値:数値の固定値を増減します。(マッスルベアー、全力攻撃など)
C値増減:クリティカル値が増減します。(かいくぐり、クリティカルキャストなど)
ダイス目増減(継続):威力値の出目が増減します(必殺攻撃など)
ダイス目増減(初回):威力値が増減します。クリティカル後に適用されません。(クリティカル・レイなど)
ダイス目固定:片方のダイス目を固定します(薬師道具セットなど)
                            
上矢印・下矢印のボタンでバフ欄を入れ替えられます

#表示/コピーについて
----
各タブの表示/コピーで、ダイスコマンドを表示、およびコマンドをクリップボードにコピーします。そのままダイスコマンドとして貼り付けて使用可能です。

個別出力:固定値・C値・命中力・回避力を、合算して出力する(例:2d6+6)か、個別に出力する(例:2d6+5+1+1-1)か選択できます
半減:ダメージを半減します。(攻撃魔法の抵抗成功時など)
